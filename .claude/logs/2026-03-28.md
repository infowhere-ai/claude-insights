### 7. Toolbar de filtros na secção pending + auto-dismiss de notificação WAITING
**O que foi feito**:
- Adição de toolbar na secção "Para Commitar" com:
  - Input de busca por path
  - Botões de filtro por status git (M mod / A add / D del / ? new / R ren) — apenas os presentes
  - Botões de filtro por extensão (.py / .html / .json…) — apenas quando há mais de uma extensão
  - Estado dos filtros persistido em `localStorage` por projecto
- Auto-dismiss da notificação "Claude is waiting for your input":
  - Aparece quando hook `Notification` dispara (state: waiting)
  - Some ao fim de 20 segundos no frontend se estado ainda for `waiting`
  - Ao expirar, limpa também o badge WAITING (state → idle)
  - Cancela-se se entretanto chegar update SSE que limpe a notificação

**Arquivos modificados**:
- `static/index.html` — toolbar pending, `_handleNotifTimer`, `notifTimers`, filtros de status/ext

**Resultado**: Toolbar de filtros funcional com persistência; notificação WAITING auto-desaparece após 20s
